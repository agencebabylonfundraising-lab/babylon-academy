# collecte.ai

> Maîtrise l'art de la conversation.

## About
collecte.ai is an AI-powered training platform for door-to-door fundraisers. Learn the craft, practice with AI donor simulations, get certified.

## Features
- 4-module curriculum (Fundamentals → Objections → Profiles → Closing)
- AI simulation: practice with 5 different donor profiles
- Gamification: XP, levels, streaks, leaderboard
- "BABYLON Certified Fundraiser" certification
- Live coaching sessions

## Tech Stack
- Frontend: HTML/CSS/JS (migrating to Next.js)
- AI Simulation: planned OpenAI/Anthropic integration
- Backend: Node.js (planned)

## Pages
- `/` — Landing + login
- `/dashboard` — Learning dashboard
- `/simulation` — AI donor simulation

🚧 MVP built — AI backend + real user accounts pending
